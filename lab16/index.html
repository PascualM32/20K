Lab 16: JSON and APIs

Challenge

For lab 16, we were tasked with the challenge to process JSON from an API, eventually creating a web page that outputs XKCD comics from the API. We also had to read through the XKCD API docs and then test the endpoint in our browsers. We were also tasked with creating a Javascript file, using AJAX to retrieve our data, storing it in a variable named comicObj. We then had to process our parts within our callback function, adding them to our webpage with the appropriate tags.
